# blog

#### 介绍
**个人博客**
基于Spring Boot的一个小而美的个人博客

#### 技术架构
后端:Spring Boot + JPA + thymeleaf模板
数据库:Mysql
前端UI:Semantic UI 框架

#### 工具与环境
- IDEA
- Maven 3
- JDK8
